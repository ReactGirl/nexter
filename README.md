## The stack
- [x] React 16.6.3
- [x] Webpack 4.19.1
- [x] Typescript
- [x] [React-hot-loader 4.6.3 ](https://github.com/gaearon/react-hot-loader)

To run the project:

    1: clone or download the repository
    2: run yarn
    3: run yarn start
